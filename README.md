🏠 Real-Estate-Market-Management-System-in-Albania

📖 A full-stack web application developed using React, Node.js, Express.js, and MySQL for managing the Albanian real estate market.

📌 Project Overview

The Real Estate Market Management System is a full-stack web application designed to simplify the management of real estate properties in Albania.

The application allows users to register, authenticate, manage properties, track transactions, and monitor property price history through an intuitive web interface backed by a relational MySQL database.

The project demonstrates the integration of a modern React frontend with a Node.js and Express backend while storing and managing data in MySQL.
✨ Features

- Secure User Registration and Login
- Role-Based Access Control (Admin, Agent, Auditor, Citizen)
- Property Management (Create, Read, Update and Delete)
- Property Zone Management
- Transaction Recording
- Automatic Price History Tracking using MySQL Triggers
- Responsive User Interface
- Relational Database Design

🛠️ Technologies Used
- React.js
- Node.js
- Express.js
- MySQL
- HTML/CSS/JavaScript
