🏠 Real-Estate-Market-Management-System-in-Albania

📖 A full-stack web application developed using React, Node.js, Express.js, and MySQL for managing the Albanian real estate market.

✨ Features
- User Authentication (Login & Registration)
- Role-Based Access Control
- Property Management (CRUD Operations)
- Transaction Management
- Price History Tracking
- MySQL Database Integration

🛠️ Technologies Used
- React.js
- Node.js
- Express.js
- MySQL
- HTML/CSS/JavaScript

  📂 Database
- MySQL Database
- Triggers for Price History Tracking
- Relational Database Design
